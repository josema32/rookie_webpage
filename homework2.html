Three New Tags


< caption > Caption Tag < /caption >

The Caption tag allows the developer to assign a title to a table.

Directions

  1. Open a new line under the table opening tag.
  2. Type caption opening and closing tags.
  3. Between the tags type the desired table name.
Class Table
Class1Class2

The code that produced the output above looks like this:

< table border="1">
< caption >Class Table< /caption >
< tr>
< td >Class1< /td >< td >Class2< /td >
< /tr >
< /table >

< div > Div Tag < /div >

Division of Page Content. Specifies a logical division within a document.
Use it to separate or identify chunks of content that are not otherwise distinguished naturally using other tags.

Directions

  1. Within a paragraph type opening a closing div tags between the text you wish to highlight.
  2. Once the tags contain the desired text, CSS attributes can be used to further visualize that section.
  3. Whole paragraphs can be contain within this tag.

This text is not enclosed into < div > tags.

This text has been enclosed into < div > tags.

Type this CSS code into your head section...

< style type="text/css" >

}
< /style >

< marquee > Marquee Tag < /marquee >

Marquee is used to scroll text across the screen.

Directions

  1. Type the marquee opening and closing tags within a desired location in the body.
  2. Type or paste the text to be scrolled between the tags.

The following words are placed between a marquee tag:

HELLO WORLD!

The code that produced the output above looks like this:

< p >The following words are placed between a marquee tag:< /p >
< marquee >HELLO WORLD!< /marquee >
